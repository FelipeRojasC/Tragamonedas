# Tragamonedas

## Descripción del Proyecto
Este proyecto es una simulación de un juego de Tragamonedas desarrollado en Java. El sistema permite a un usuario ingresar un saldo inicial y realizar apuestas. En esta versión refactorizada, se ha la clase `Box` y trasladando su lógica de negocio directamente a la implementación principal del servicio para reducir el acoplamiento innecesario.
